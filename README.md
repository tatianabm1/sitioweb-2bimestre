# Sitio web educativo Rediseñado con Bootstrap
El sitio web educativo que se creó para la tarea del primer bimestre fue rediseñado específicamente para el diseño centrado en el usuario usando los principios de diseño web como la usabilidad y accesibilidad.
El framework utilizado para el rediseño fue Bootstrap 5.3.3 (CDN) junto con un archivo de estilos CSS propio creado para este rediseño.
